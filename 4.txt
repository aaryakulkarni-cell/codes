Write a PL/SQL code block to calculate the area of a circle for a value of radius varying from 5 to 
9. Store the radius and the corresponding values of calculated area in an empty table named areas, 
consisting of two columns, radius and area. 
Note: Instructor will frame the problem statement for writing PL/SQL block in line with above 
statement. 

Circle - 

create database ass5;
use ass5;

create table circle1(radius int(20),area float(20),diameter int(20),circumference float(20));

drop procedure if exists circlep;

delimiter //

create procedure circlep()
begin
declare radius int;
declare diameter int;
declare area float;
declare circumference float;
declare pi float default 3.14;

set radius:=5;

cloop: LOOP
set diameter:= 2*radius;
set area:= pi*radius*radius;
set circumference:=2*pi*radius;

select radius,diameter,area,circumference;
insert into circle1(radius,area,diameter,circumference) values (radius, area, diameter,circumference);
set radius=radius+1;
if radius>9 then
leave cloop;
end if;

END LOOP cloop;
end;
//

delimiter ;

call circlep();

select *from circle1;
